# Blocks Design Framework

## What is Blocks Design Framework?
 
**The Blocks Design Framework is the capstone project for The University of Sydney COMP5703 Semester 2, 2023**

This project is a web design framework and it is designed to be used by the University of Sydney, but it can be easily modified to be used by other universities or higher education institutions.

## Which is included in the Blocks Design Framework?

The **Blocks Design Framework** consists of two parts: **Components** and the **Pages**.

## How to use the Blocks Design Framework?

Simply get started by clicking [here](accordion) or browse the **Components** documentation to learn more.
